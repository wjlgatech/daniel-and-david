# Curriculum

Two tracks, one method: **learn a principle, then apply it on a live venture, and let the
venture teach you back.** No worksheets-for-their-own-sake. Everything points at building
something real.

| Track | For | Focus |
|---|---|---|
| 👦 [Daniel — age 11](daniel-age-11/) | the builder | ship, price, sell, measure, read a P&L, code with an AI pair |
| 🧒 [David — age 6](david-age-6/) | the apprentice | count, create, greet the customer, learn that good work serves people |

## How the tracks work

Each track is a set of **quests**. A quest is small, doable in a sitting or two, and ends
with *something real you made or measured.* Quests level up: each one assumes the last.

The ventures (starting with [KC Matchday Basecamp](../../ventures/kc-matchday-basecamp/)) are
the **field** where quests get applied. Reading about margin is a worksheet. Pricing a real
kit, selling it, and counting what's left over is a *quest*.

## The three strands every quest touches

1. **Build** — make a real thing (a sign, a page, a kit, a line of code).
2. **Business** — understand value, price, margin, customer, demand.
3. **Character** — honesty, courage, service, generosity (the [values](../principles/values.md)).

## For parents / mentors

- **Keep it real and small.** A finished tiny thing beats an abandoned big thing.
- **Let them feel the market.** The customer's "yes" or "no" teaches more than you can.
- **Protect the joy.** This is a workshop, not a pressure cooker. The number is a compass,
  never a whip. (See [mission.md](../vision/mission.md).)
- **Model the values.** They're learning how you treat people more than what you say.
