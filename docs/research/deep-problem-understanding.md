# Research — deep problem / pain understanding (first-principles)

*Synthesis behind [Pain2Gain](../principles/pain2gain.md). Reuse before re-researching root-cause / needs / depth.*

## Key findings
- **Multi-dimensional pain** — biopsychosocial (Engel 1977) + **spiritual** (Sulmasy 2002). Translate to: physical/operational · psychological · behavioral · relational · organizational · meaning/spiritual.
- **Causal depth** — **5W1H scopes, 5-Why finds cause** (Ohno; sequential, not interchangeable). 5-Why pitfalls: single-thread, confirmation bias, knowledge ceiling → fix with **Three-Legged 5-Why** (occurrence/detection/systemic), **Ishikawa fishbone** (force branches), and **evidence at every link**. Pearl's hierarchy: associational → interventional → counterfactual/mechanistic.
- **Leverage** — Meadows' 12 points: most fixes hit weak levers (parameters); transformative = rules/goals/**paradigm**. First-principles test: *"would the pain persist if every workaround worked?"* → yes = symptom.
- **Need framing** — JTBD (functional/emotional/social; Ulwick outcomes), Value Proposition Canvas (pains/gains/jobs), Kano (broken **Basics** hurt most), SDT (autonomy/competence/relatedness — strong cross-cultural evidence), Max-Neef (9 needs; beware "violators"). Maslow = heuristic only (weak empirical support).
- **Distribution** — burden ≈ **prevalence × severity × duration** (DALY-style) + spatial/temporal. **N=1 thick description (Geertz)** beats large-N survey for *understanding* (mechanism); survey *measures* after.
- **Depth ladder** (don't build before rung 4; transformative at 5–6): anecdote → survey → correlation → causal chain → mechanism → first-principle.
