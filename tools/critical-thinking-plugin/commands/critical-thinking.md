Apply the **5W1H critical-thinking grid** to the following target:

> $ARGUMENTS

If no target was given above, ask the user for one (a claim, plan, product, venture idea, or
decision), then proceed.

Point all six question-words at the target. Pick the 3–5 questions per word that bite hardest —
always include the uncomfortable ones. Prefer hard truths over reassurance.

- **WHO** — benefits / is harmed / decides / is most affected / is missing (esp. people who can't speak for themselves)?
- **WHAT** — strengths & weaknesses / another perspective / the counter-argument / best & worst case / what's in the way?
- **WHERE** — in the real world / similar precedents / most need / could become a problem / where to get help?
- **WHEN** — acceptable vs not / benefits or harms society / right time to act / how we'll know we succeeded?
- **WHY** — is it worth solving / relevant / why hasn't it been done / is there a need today? (keep asking "why")
- **HOW** — does it work / how do we KNOW it's true (verify, don't assume) / approach safely / change it for good?

Then return:

1. **Target** — one line.
2. **Sharpest findings** — 3–6 bullets, each tagged with its word and `(risk | opportunity | assumption | unknown)`.
3. **The uncomfortable question** — the one most tempting to skip, answered honestly.
4. **Verdict** — `proceed` / `proceed-with-changes` / `stop`, with the single most important reason, cross-checked against the four-question values test (true? legal & word kept? serves? proud?).

Flag any values/legal judgment call for a human — do not self-approve it.
