# Critical Thinking (5W1H)

A structured lens for interrogating *any* target — a claim, a plan, a product, a decision, a
piece of information, a PR — before acting on it.

## When to use this

When the task involves **judgment, not just execution**: before a money/customer decision,
before trusting a factual or AI-generated claim (*verify, then trust*), before selling, or when
reviewing a plan for what's *missing*. Skip it on trivial, reversible mechanical work.

## How to apply it

Point all six question-words at the target. Pick the 3–5 per word that bite hardest; always
include the uncomfortable ones.

- **WHO** — benefits / harmed / decides / most affected / should be consulted / is missing?
- **WHAT** — strengths & weaknesses / another perspective / counter-argument / best & worst case / what's in the way?
- **WHERE** — real-world sightings / precedents / most need / could be a problem / where to get help?
- **WHEN** — acceptable or not / benefits or harms society / best time / how we'll know we succeeded / might change?
- **WHY** — worth solving / relevant / why not done already / a need today? (keep asking "why")
- **HOW** — works / similar to known / could harm / how do we KNOW it's true / approach safely / change for good?

## Output format

1. **Target** — one line.
2. **Sharpest findings** — 3–6 bullets tagged with the word and `(risk | opportunity | assumption | unknown)`.
3. **The uncomfortable question** — answered honestly.
4. **Verdict** — proceed / proceed-with-changes / stop + the single most important reason.

Cross-check the verdict against a four-question values test (true? legal & word kept? serves?
proud?). Flag values/legal judgment calls for a human — don't self-approve them. An honest
"stop" beats a rationalized "proceed."
