## なぜこれが存在するのか

多くの人は仕事を*探す*ように教わります。私たちは価値を*生み出す*ことを学びます — 人が本当に必要とするものを
つくり、誠実に行い、返ってくるものをよく管理する。子どもはこれを講義ではなく、**AI チームとともに本物を
一つ、最初から最後までつくる**ことで学びます。[`docs/vision/mission.md`](docs/vision/mission.md) と
[変革の理論](docs/vision/theory-of-change.md) を参照 — 因果モデルと、私たちが実際に追う指標
（*子ども 1 人・1 か月あたりの「自立したつくり手の証拠」*、お金やスターではなく）。

> **より深い狙い。** 私たちは祝福するためにつくります。ここのどのベンチャーも、いくら稼ぐかだけでなく、
> 誰に仕え、他者に何を可能にするかで測られます。[`docs/principles/values.md`](docs/principles/values.md)。

> **年長の学習者と保護者へ：** 長期の*ベンチャー・トラック* — $1M→$10B のマイルストーンのはしご — も
> ありますが、意図的に見出しには**しません。** 約束ではなく羅針盤で、私たちが前面に出す能力のはしごの下、
> [`docs/vision/milestones.md`](docs/vision/milestones.md) にあります。

### カリキュラム

一つの方法を、**どの年齢にも** — ドリルではなく本物づくりに結びつけて。

**子どもたちへ**（本物のベンチャーに接続）：

- 👦 **[Daniel — 11 歳](docs/curriculum/daniel-age-11/)** — つくり、値付けし、売り、測り、損益を読む。AI の
  相棒とコードを書く。（いまは [AI ビルドキット](docs/curriculum/daniel-age-11/build-kit.md) を率いる：つくる →
  メンターに見せる → 大人とともに売る。）
- 🧒 **[David — 6 歳](docs/curriculum/david-age-6/)** — 数え、看板を描き、お客さんを迎え、*人のために良いものを
  つくること*こそが全部だと学ぶ。

**大人たちへ** — 大人専用の [回復の道](docs/vision/isaiah-35-restoration.md)、4 週間の
[回復ループ](docs/curriculum/adult-restoration/) として（*肌は老いても心は若い*人も歓迎）。同じ Builder
Loop を、一つの人生を立て直すことに向けて：

- 👩‍👧 **ひとり親** — 本物の痛点を一つ見つけ、AI チームと小さな有料プロダクトを出し、最初の*「はい」*を得る。履歴書より回復。
- 👨‍💻 **揺さぶられた専門職**（AI がちょうど変えたその職務） — 苦労して得た知見をプロダクトに；かつて一部署が要った仕事を AI チームを率いて。
- 👵 **熟練のメンター**（教師・管理職・退職者） — 一生分の知恵を、人に仕える（そして売れる）何かに。遅くないという証拠。
- 🚗 **若き挑戦者**（ドライバー・接客・ギグワーカー） — 副業をくり返せるプロダクトへ；利益・需要・レバレッジを学ぶ。

> どの大人トラックも同じエンジンを四つの R で回します — **回復 · 贖い · 再生 · 復活**（イザヤ書 35 章）。*設計中。*

### 原則

私たちの働き方 — 最高の AI ネイティブ企業と同じく：

### 🛠️ 開発者・貢献者・創設ファミリー向け — ワンクリック・セットアップ

```bash

./scripts/setup.sh        # ← 一つのコマンド：スクリプトを実行可能に + すべてのガードレールを実行
```

`setup.sh` は何度実行しても安全です。[Claude Code](https://claude.ai/code) を検出し、**このリポジトリ同梱の
AI ツールキット**へ案内します — ワンクリック導入、グローバル設定不要：

- **`/check`** — CI と同じやり方で五つのガードレール（links · toolkit · status-truth · registration-safety · webapp）を全部実行。
- **`critical-thinking` プラグイン**（5W1H 格子をコマンド + スキル + フックに）。このリポジトリ自身のマーケットプレイスから導入：
  ```text
  /plugin marketplace add wjlgatech/daniel-and-david
  /plugin install critical-thinking@daniel-and-david
  ```

その後：ローカルでランディングページを開き（`open apps/web/public/index.html`）、**[AGENTS.md](AGENTS.md)** と
**[.claude/README.md](.claude/README.md)** を読み、PR の前に `./scripts/check.sh` を実行。

## 私たちが守る基準

<p align="center"><img src="docs/assets/four-question-test.svg" alt="The 4-Question Test: Is it true? Is it legal and is my word kept? Does it serve someone? Would I be proud to explain it?" width="92%"></p>

- **常に合法で正直に。** 1 号ベンチャーに明確なコンプライアンス・ゲートと「やらないこと」リストがあるのには理由があります。近道ではなく、信頼で勝ちます。
- **小さく、本物を出す。** 動くものは完璧な計画に勝ります。
- **つくりながら教える。** 6 歳が*なぜ*をつかめないなら、説明が足りないということ。
- **育ちながら分かち合う。** 寛容さは後付けではなく、モデルに組み込まれています。
